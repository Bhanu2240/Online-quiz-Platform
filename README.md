# 🧠 Online Quiz Platform

A fully responsive **Online Quiz Application** built using **HTML, CSS, and Vanilla JavaScript**.
This project demonstrates core front-end skills including DOM manipulation, event handling, and clean code structure.

---

## 🚀 Features

* ✅ Start Quiz screen
* ✅ Multiple choice questions (MCQs)
* ✅ One question at a time
* ✅ Timer for each question
* ✅ Instant answer validation (correct/wrong highlight)
* ✅ Score calculation
* ✅ Final result with percentage
* ✅ Restart Quiz option
* ✅ High score stored using **localStorage**
* ✅ Fully responsive design (Mobile + Tablet + Desktop)
* ✅ Clean and modern UI

---

## 🛠️ Tech Stack

* **HTML5** – Structure
* **CSS3** – Styling & Responsive Design (Flexbox/Grid)
* **JavaScript (ES6)** – Logic & Interactivity
